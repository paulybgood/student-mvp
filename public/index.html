Things You Can Do:

  • Create a username so you can save your to-do lists and come back to them later
  • Create as many to-do lists as you want
  • Create as many tasks as you would like
  • Delete to-do lists and tasks when you are done with them

How To Get Started:

  • Create and Register a username in the top-right corner of the page
  • Create your first to-do list by clicking the "Create To-Do List" Button
  • View your newly created to-do list by clicking on the "View List" button
  • Create a task by clicking on the "Create New Task" button
  • Congratulations! You just successfully created your list. Enjoy!
